Q-2 What are tags and attributes in HTML?

=> Tags are used to define elements or structures within a document.
   Tags are enclosed in angle brackets (< >).
   Example of tag:<p>This is a paragraph.</p>

   Attributes provide additional information about HTML elements.
   They are always included in the opening tag and are written as name-value pairs.
   Attributes usually come after the element name and are separated by spaces.
   Example of attributes: <a href="https://www.example.com">Visit Example.com</a>
   In this example, href is an attribute of the <a>(anchor)tag, and it specifies the hyperlink destination.

Q-5 What are different types of lists in HTML?

=>HTML provides several types of lists to organize and structure content on a web page. The main types of lists in HTML are:
  1)Ordered Lists (<ol>):
   Ordered lists are used to present items in a numbered sequence.
   Each list item is wrapped in <li> (list item) tags.
Example:<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
  </ol>
   
   2)Unordered Lists (<ul>):
   Unordered lists are used to present items in a bulleted or marked list.
   Each list item is also wrapped in <li> tags.
Example:<ul>
  <li>Apples</li>
  <li>Oranges</li>
  <li>Bananas</li>
  </ul>
   
   3)Description Lists  (<dl>):
   Description lists are used to define terms and provide descriptions for each term.
   Each term is defined using the <dt> (definition term)and the description is provided with the <dd> (definition description) tag.
Example:
   <dl>
  <dt>HTML</dt>
  <dd>Hypertext Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
  </dl>

Q-9 How is Cell Padding different from Cell Spacing?

=>cell padding and cell spacing are attributes that can be applied to the <table> element to control the layout and spacing of cells within the table.

 1)Cell Padding:

   Attribute Name: cellpadding
   Description: Cell padding controls the space between the content of a table cell and the cell's border. It adds space inside each cell.
   Example:<table cellpadding="10">
       <!-- Table content -->
        </table>

 2)Cell Spacing:

   Attribute Name: cellspacing
   Description: Cell spacing controls the space between adjacent cells in the table. It adds space between the borders of neighboring cells.
   Example:<table cellspacing="5">
           <!-- Table content -->
            </table>

   cellpadding="10" adds 10 pixels of padding inside each cell.
   cellspacing="5" adds 5 pixels of space between the borders of adjacent cells.

Q-11 What is the difference between a block-level element and an inline element?

=>1)Block-level elements:
  These elements create a "block" or a rectangular box in the layout.
  They typically start on a new line and take up the full width available, extending as far as the container allows.
  Examples of block-level elements include <div>, <p>, <h1> to <h6>, <ul>, <ol>, and <li>.
  
  2)Inline elements:

  These elements do not create a new "block" and only take up as much width as necessary.
  They do not start on a new line and flow within the content.
  Examples of inline elements include <span>, <a>, <strong>, <em>, <img>, and <br>.

Q-13 What is the use of an iframe tag?

=>The <iframe> (inline frame) tag in HTML is used to embed another HTML document within the current document. It allows you to include external content such as a web page, a video, a map, or any other HTML document while keeping them independent of each other. The content inside the <iframe> is displayed within a rectangular frame on the page.

Example:<iframe width="560" height="315" src="https://www.youtube.com/embed/7nhfTRr94P4?si=PJ3JaLK3_Pmj5yxp" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>
